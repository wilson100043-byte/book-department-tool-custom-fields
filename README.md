# 書單已購買與系所比對工具｜自訂欄位用途版

純前端單檔 HTML 工具，使用 XLSX.js 在瀏覽器本機處理 Excel，不需要後端，也不需要 Node.js。

## 使用方式

1. 開啟 `index.html`。
2. 上傳出版社書單 Excel，選擇工作表。
3. 用下拉選單指定出版社書單欄位用途。
4. 若有已購買書單，另外上傳並指定已購買 ISBN / 查重欄位。
5. 貼上 AI 回傳 CSV，格式固定為：

```csv
學校名稱,系所名稱,關鍵字
淡江大學,財務金融學系,財務金融;金融;投資;corporate finance
```

6. 產生比對結果 Excel。
7. 若只需要系統上傳格式，可直接使用「系統上傳格式（獨立功能）」區塊，不需要先跑系所比對。

## 主要特色

- 不寫死 Wiley 或任何出版社欄位名稱。
- 出版社原始欄位不會被修改。
- 輸出 Excel 保留原始欄位與原始順序，只在最後新增結果欄位。
- 系所比對只使用使用者指定的書名、主題、子主題、內容簡介、目次欄位。
- 已購買查重使用指定 ISBN / 查重欄位，並先 normalize ISBN。
- 系統上傳格式是獨立功能，依使用者指定欄位直接產生。

## 輸出新增欄位

- 對應系所
- 命中關鍵字
- 系所標記狀態
- 是否已購買
- 購買比對值
- 購買比對來源
- 已購買日期
